## Highlights

- Balanced and multidimensional tree implementations.
- Skip-list and clustering exercises.
- Clear title-based project organization for comparative review.

## Getting started

1. Create a Python 3.10+ environment.
2. Run `python -m pip install -e .` for the packaged source where applicable.
3. Review each project directory independently; course-specific inputs may vary.

## Portfolio note

The repository is a portfolio of independent coursework projects rather than a single production library.
